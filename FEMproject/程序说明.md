本项目为自研的有限元程序基础库，以及几个程序实例组成。此程序依赖的第三方库为Eigen3.4.0。有限元基础库提供了有限元常用的计算函数和通用父类，以方便有限元编程。
# 基础库架构
此库将有限元程序划分为三层架构：

+ 单元类
+ 物理场类（包含一个或多个单元类）
+ 程序总体数据类（包含一个或多个物理场类）

## 单元类

每种单元使用一个单元类表示，这里所说的一种单元指的是一种材料在某个物理场下的一种单元。
### 数据

+ 单元基本信息：名称(_name)、类型(_types)、节点个数(_nNode)、单元数量(_nElement)
+ 单元编号：_eleIds数组，以及与材料ID的映射(_eleMatIDMap)
+ 自由度信息：自由度名称数组(_dispNames)、未知量个数(_nVar)、广义位移个数(_nDisp)
+ 维度信息：全局空间维度(_dim)、全局坐标维度(_nCoor)
+ 计算结果：实时计算结果(_result)、各单元结果数组(_results)、保存标志(_bSaveResult)
+ 材料参数：单元子程序所需的材料参数名称数组(_paramNames)
+ 所属关系：所属物理场指针(_phyData)、所属有限元空间指针(_femData)
+ VTK单元类型(_vtkCellType)

计算结果通过EleSubResult结构体存储，包含：单元刚度矩阵(estif)、质量矩阵(emass)、阻尼矩阵(edamp)、外力向量(eload)、节点编号(nodeIds)。

后处理结果通过uResult结构体存储，包含：单元结果映射表(eleResult)和节点结果映射表(nodeResult)。

### 核心虚函数

#### run函数（单元子程序）
```cpp
virtual EleSubResult& run(const std::vector<double>& r, 
                          const std::map<std::string, std::vector<double>>& coef, 
                          const std::map<std::string, double>& matParams);
```
计算单元刚度矩阵、质量矩阵、阻尼矩阵和外力向量。派生类必须实现此函数。

参数说明：
- r：节点坐标数组
- coef：计算系数（如形函数值、形函数偏导等）
- matParams：材料参数映射表

#### uEle函数（后处理程序）
```cpp
virtual uResult uEle(const std::vector<double>& r, 
                     const std::map<std::string, std::vector<double>>& coef, 
                     const std::map<std::string, double>& matParams);
```
根据计算结果计算应力、轴力等物理量。派生类可选实现此函数。

### 派生类实现要求

派生ElementBase类时，需要：
1. 在构造函数中设置单元基本信息（_name、_nNode、_dispNames等）
2. 实现run函数，完成具体单元的计算逻辑
3. 可选实现uEle函数，完成后处理计算

示例派生类：Truss2D（二维桁架单元）、Truss3D（三维桁架单元）等。

## 物理场类

物理场类(PhyFieldData)包含一个或多个单元类，负责该物理场的计算和后处理。在此类下实现以下程序：
+ eProgram程序，计算总刚矩阵和右端项
+ uPhy后处理程序，根据计算结果计算节点和单元的物理量

此类实例中存储以下数据：

+ 场基本信息：名称(_name)、结果格式(_resForm)、自由度名称(_dispNames)
+ 结果名称：节点结果名称(_nodeResNames)、单元结果名称(_eleResNames)
+ 自由度信息：总自由度数(_kVar)、自由度个数(_dof)、2类边值自由度(_dof2)
+ 边界条件：第一类边界条件(_nodeBC1s)、第二类边界条件(_nodeBC2s)
+ 关联的场：关联的物理场数组(_assPhys)
+ 有限元空间数据：所属的有限元空间指针(_femData)
+ 单元子程序：单元类型数组(_eleSubs)
+ 方程系统：节点规格数(_ida)、方程系统(_equSys)、等式个数(_neq)
+ 计算结果：节点结果映射表(_nodeRes)、单元结果映射表(_elemRes)

### 核心虚函数

#### eProgram函数（计算程序）
```cpp
virtual int eProgram() { return eProgram_el(); }
```
计算总刚矩阵和右端项。默认使用椭圆方程求解，派生类可选实现此函数。


### 主要成员函数

#### eProgram_el函数（椭圆方程求解）
```cpp
int eProgram_el();
```
求解线性椭圆方程的算法。此函数已实现，负责组装刚度矩阵、应用边界条件。

#### uPhy函数（后处理程序）
```cpp
virtual int uPhy();
```
根据计算结果，计算节点和单元的物理量（如应力、应变等）。派生类可选实现此函数。

#### 初始化相关
- `setNPts(int n)`：设置节点个数，并初始化相关数值
- `initMatrix()`：根据边界条件及单元初始化矩阵

#### 单元管理
- `addEleSub(ElementBase* eleSub)`：添加单元类型，返回单元序号

#### 边界条件设置
- `addBoundary(int nodeId, int typeId, double val, int iDof)`：添加边界条件
- `setFirstBoundry(int nodeId, double val, int iDof)`：设置第一类边界条件
- `setSecondBoundry(int nodeId, double val, int iDof)`：设置第二类边界条件

#### 求解器
- `solve()`：求解矩阵方程

#### 结果获取
- `getCoef1(std::vector<int> nodeIds)`：获取计算系数
- `getCoef(std::vector<int> nodeIds)`：获取计算系数
- `getNodeDisps(const std::vector<int>& nodeIds)`：获取节点位移
- `getNodeDisps(const std::vector<int>& nodeIds, int iDof)`：获取指定自由度的节点位移

## 程序总体数据类

程序总体数据类(FEMData)包含一个或多个物理场类，负责整个有限元程序的数据管理和计算流程控制。在此类下实现以下程序：
+ caculate计算程序，控制整个计算过程
+ main主程序，程序的入口点

此类实例中存储以下数据：

+ 网格信息：节点数(_nPts)、节点坐标(_nodes)、单元节点(_eleNodes)、单元类型(_eleTypes)、单元数量(_nEles)
+ 单元索引：单元指针(_elePt)、总单元数(_nElem)、单元材料号(_eleMateIds)
+ ID映射：节点ID映射(_nodeIdMap)、单元ID映射(_eleIdMap)
+ 物理场：物理场数组(_phyDatas)
+ 材料参数：材料参数数组(_mateParams)、材料名称数组(_mateNames)
+ 时间参数：时间步长(_dt)、最大时间(_tMax)
+ 维度：空间维度(_dim)

### 核心虚函数

#### caculate函数（计算程序）
```cpp
virtual int caculate() { return -1; }
```
控制整个计算过程。派生类必须实现此函数。

#### main函数（主程序）
```cpp
virtual int main() { return -1; }
```
程序的入口点。派生类必须实现此函数。

### 主要成员函数

#### 节点管理
- `setNPts(int n)`：设置网格点的个数
- `addNode(int id, double x, double y, double z)`：添加节点
- `addNodeEnd()`：节点添加结束后，调用此函数

#### 单元管理
- `addEle(int id, const std::vector<int>& nodeIds, const std::string& eleType)`：添加单元
- `setEleMateId(int eleId, int id)`：设置单元材料号
- `setEleMateByName(int eleId, const std::string& name)`：通过材料名称设置单元材料号

#### 材料管理
- `addMate(const std::map<std::string, double>& matParam, const std::string& name)`：添加材料参数，返回材料号
- `getElemMatParams(int eleID, ElementBase* ele)`：获取单元的材料参数

### 派生类实现要求

派生FEMData类时，需要：
1. 在构造函数中设置网格信息（节点、单元等）
2. 实现caculate函数，控制整个计算过程
3. 添加物理场对象到_phyDatas数组


示例派生类：根据具体应用问题实现，如结构力学问题、热传导问题等。
